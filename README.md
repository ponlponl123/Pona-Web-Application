This is the web application for Pona, a feature-rich Discord bot. This frontend provides a user-friendly interface for interacting with the bot and visualizing data. As a hobby project, the code is a continuous work-in-progress, crafted with ❤️ for the community.

## Key Features & Technologies

This project is built with a modern tech stack to deliver a responsive and interactive experience:

- **Framework**: [Next.js](https://nextjs.org/) with [React](https://react.dev/) for server-side rendering and a robust application structure.
- **UI Components**: A beautiful and accessible UI built with [NextUI](https://nextui.org/) and styled with [Tailwind CSS](https://tailwindcss.com/).
- **Real-time Updates**: [Socket.IO](https://socket.io/) is used for live communication between the client and the server.
- **Data Visualization**: Interactive charts are rendered using [Recharts](https://recharts.org/), and maps are displayed with [Leaflet](https://leafletjs.com/).
- **Animations**: Smooth and engaging animations are powered by [Framer Motion](https://www.framer.com/motion/).
- **Discord Integration**: Leverages [Discord.js](https://discord.js.org/) for interactions with the Discord API.
- **HTTP Requests**: [Axios](httpss://axios-http.com/) for making promises-based HTTP requests.
